# My Gin App

My first [gin](https://gin-gonic.github.io/gin/) app.

## Installation

Install packages:

```sh
go install # or from anywhere with: go install github.com/s2t2/my_gin_app
```

## Usage

Run the app:

```sh
$GOPATH/bin/my_gin_app # or just... my_gin_app
```

Then view in browser at localhost:8080.
